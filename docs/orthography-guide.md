# Orthography Guide: Writing the Ohlone Languages

This guide describes the modern orthographic conventions used for each Ohlone variety, based on published dictionaries and community materials.

## Mutsun Orthography

**Primary source:** Warner, Butler & Geary (2016), *Mutsun-English English-Mutsun Dictionary*

### Consonants

| Letter | IPA | Description | Example |
|--------|-----|-------------|---------|
| c | /tʃ/ | like English "ch" in "church" | `caahi` "barn owl" |
| d | /d/ | as in English (loanwords) | `diyos` "God" |
| h | /h/ | as in English "hat" | `haahe` "run away" |
| k | /k/ | as in English "key" | `kaa` "daughter" |
| l | /l/ | as in English "let" | `laake` "rise" |
| **L** | /lʲ/ | palatalized L, like "tell you" | `Luohu` "yearling calf" |
| m | /m/ | as in English "map" | `maahi` "close, cover" |
| n | /n/ | as in English "net" | `naaru` "turnip" |
| **N** | /ɲ/ | like Spanish ñ | `Notko` "be short" |
| p | /p/ | as in English "pet" | `paaka` "shell" |
| r | /ɾ/ | single tap/flap (like Spanish r) | `raakat` "name" |
| s | /s/ | as in English "sun" | `saake` "gather pinenuts" |
| **S** | /ʂ/ | retroflex S (tongue curled back) | `Saanay` "nearby" |
| t | /t/ | as in English "top" | `taacin` "river rat" |
| **T** | /ʈ/ | retroflex T (tongue curled back) | `Taakampi` "bring, carry to" |
| **ts** | /ts/ | as in English "cats" | `tsayla` "lie face up" |
| **tY** | /tʲ/ | palatal stop | `tYottYoni` "holly berry" |
| w | /w/ | as in English "wet" | `waaha` "scratch" |
| y | /j/ | as in English "yes" | `yaase` "eat" |
| **ʼ** | /ʔ/ | glottal stop (catch in throat) | `-ʼa` |

### Vowels

| Letter | IPA | Description |
|--------|-----|-------------|
| a | /a/ | as in "father" |
| e | /e/ | as in "met" |
| i | /i/ | as in "machine" |
| o | /o/ | as in "wrote" |
| u | /u/ | as in "rule" |

### Key Rules

1. **Doubling = length**: `aa` is long /aː/, `tt` is geminate /tː/
2. **Capital letters are separate sounds**: `s` ≠ `S`, `l` ≠ `L`, `n` ≠ `N`, `t` ≠ `T`
3. **Dictionary order**: a, c, d, e, h, i, k, l, L, m, n, N, o, p, r, s, S, t, T, ts, tY, u, w, y, ʼ
4. **Stress**: Generally on the first syllable

**Primary sources:** Kaufman (2008); Kroeber (1904); academic orthography

| Letter | IPA | Description | Example |
|--------|-----|-------------|---------|

| ţ | /ts/ | as in "cats" | |
| ŝ | /ʃ/ | alternative for postalveolar | |
| x | /x/ | as in Scottish "loch" | |
| ʼ or ʔ | /ʔ/ | glottal stop | |

### Key Rules

1. **Stress**: Generally word-initial (first syllable)
2. **Vowel length**: Marked by doubling
3. **Consonant length**: Also marked by doubling

## Chochenyo Orthography

**Primary sources:** Muwekma Ohlone Tribe materials; Harrington field notes

### Letter Values (differ from English!)

| Letter | IPA | Unlike English... |
|--------|-----|-------------------|
| **c** | /tʃ/ | Not /k/ — sounds like "ch" |
| **s** | /ʃ/ | Not /s/ — sounds like "sh" |
| **x** | /h/ | Not /ks/ — sounds like "h" |
| **T** | /ʈ/ | Retroflex (tongue curled) |
| **ʼ** | /ʔ/ | Glottal stop |

### Vowels

Same five-vowel system as Mutsun, with Spanish-like values.

### Example Words

| Chochenyo | Meaning | Pronunciation Guide |
|-----------|---------|-------------------|
| xučyun | Berkeley shoreline area | hu-CHYUN |
| Tújŝtak | Mount Diablo ("little hill") | TUJ-shtak |
| makkin | we (inclusive) | MAK-kin |
| muwekma | the people | mu-WEK-ma |
| wolwoolum | language/speech | wol-WOO-lum |
| saleki | good | sa-LEK-i |

## OCEN Rumsen Orthography

**Primary source:** Ohlone Costanoan Esselen Nation, Louise J. Miranda Ramirez & David L. Shaul

### Letter Values

| Letter | Value | Description |
|--------|-------|-------------|
| X/x | /h/ | Sounds like H |
| C/c | /tʃ/ | Sounds like CH |
| S/s | /ʃ/ | Sounds like SH |
| T | retroflex | Curl tongue back to palate |
| R/r | /ɾ/ | Probable flap/tap |

### Stress

Penultimate (next-to-last syllable) stress: `sa-LEK-i`, `to-MA-nis`

### Example Phrases

| OCEN Rumsen | Meaning | Pronunciation |
|-------------|---------|---------------|
| Saleki Asatsa | Good Morning | sa-LEK-i a-SAT-sa |
| Saleki Itsa | Good Afternoon | sa-LEK-i IT-sa |
| Saleki Itsu | Good Evening | sa-LEK-i IT-su |
| Saleki Tomanis | Good Night | sa-LEK-i to-MA-nis |
| Lex Welel | Our Language | — |

## Typing Ohlone Characters

### Quick Reference: How to Type Special Characters

| Character | macOS | Windows | HTML Entity |
|-----------|-------|---------|-------------|
| ʼ (glottal stop) | Option+Shift+] | Alt+0700 | `&#x02BC;` |
| č | Option+v then c | Alt+0269 | `&ccaron;` |
| ş | — (use keyboard layout) | Alt+0351 | `&#x015F;` |
| ŝ | Option+6 then s | Alt+0349 | `&#x015D;` |
| ţ | — (use keyboard layout) | Alt+0355 | `&#x0163;` |
| ʔ | — (use keyboard layout) | Alt+0660 | `&#x0294;` |

### Common Mistakes to Avoid

1. **Don't use a regular apostrophe** (`'` U+0027) for the glottal stop — use `ʼ` (U+02BC)
2. **Don't use smart quotes** (`'` U+2019) for the glottal stop
3. **Don't lowercase Mutsun L, N, S, T** — they are separate sounds from l, n, s, t
4. **Don't confuse `c` with English "c"** — in Chochenyo and OCEN, `c` = "ch"
5. **Don't confuse `s` with English "s"** — in Chochenyo and OCEN, `s` = "sh"
